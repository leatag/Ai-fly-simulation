# 🌿 Цифровой Организм - 3D Симулятор Эволюции
# Digital Organism Evolution Simulator - 3D Edition

**Красивая реалистичная 3D визуализация эволюции цифровых организмов с поддержкой Panda3D!**

Beautiful realistic 3D visualization of digital organism evolution with Panda3D support!

## 🎮 Запуск 3D Версии

```bash
# Базовый запуск
python main_3d.py

# Или из терминала Linux/Mac
./main_3d.py
```

## 🎯 Управление

| Клавиша | Действие |
|---------|----------|
| **Мышь** | Вращение камеры (drag) |
| **W** | Движение камеры вперед |
| **A** | Движение камеры влево |
| **S** | Движение камеры назад |
| **D** | Движение камеры вправо |
| **Q** | Камера вверх |
| **E** | Камера вниз |
| **ESC** | Выход из симуляции |

## ✨ Возможности 3D Версии

- ✅ **Реалистичные 3D модели** - Красивые мухи с анатомическими деталями
- ✅ **Динамическое освещение** - Солнце, точечные светильники, амбиентный свет
- ✅ **Красивая территория** - Трава, деревья, неровный ландшафт
- ✅ **Анимация крыльев** - Летающие насекомые с реалистичными движениями
- ✅ **Эмоциональные цвета** - Организмы меняют цвет в зависимости от эмоции
- ✅ **Свободная камера** - Полная навигация в 3D пространстве
- ✅ **Реальная эволюция** - Все AI и генетические системы работают

## 📊 Интеграция с Симуляцией

Все основные системы симуляции работают с 3D визуализацией:

- 🧬 **Генетика** - Организмы наследуют гены и эволюционируют
- 🧠 **ИИ** - Эмоциональные состояния, обучение страху, принятие решений
- 🏠 **Поведение** - Построение убежищ, поиск пищи, размножение
- 📈 **Эволюция** - Отслеживание поколений, мутации, естественный отбор

## 🎨 Визуальные Особенности

### Модели Организмов
- Реалистичное тело с сегментами
- Голова с красными глазами
- Крылья с анимацией
- 6 ног для каждого организма
- Усики для сенсоров

- Зеленая трава с текстурными вариациями
- Деревья с кроной и стволом

- Динамические тени

### Освещение
- Солнечный свет (directional light)
- Окружающее освещение
- Точечные светильники для глубины
- Реалистичные материалы и отражения

## 🔧 Технические Детали

### Использованные Библиотеки

```python
Panda3D       - 3D графический движок
NumPy         - Математические вычисления
Pygame        - Исходные компоненты симуляции
```

### Архитектура

- `panda3d_engine.py` - Основной 3D движок
- `main_3d.py` - Запуск 3D симуляции
- `simulation.py` - Логика эволюции (совместимо с 3D)
- `config.py` - Глобальные параметры

### FPS и Производительность

- **Динамическая оптимизация** - Количество организмов влияет на FPS
- **Efficient 3D Rendering** - Использует Panda3D для оптимальной производительности
- **Real-time Evolution** - Все 2000+ симуляций выполняются в реальном времени

## 💡 Советы

- **Для лучшего наблюдения**: Переместите камеру повыше и подальше (E + S + A/D)
- **Следите за организмами**: Выбирайте интересных (красный = напуган)
- **Изучайте поведение**: Смотрите как организмы строят убежища и размножаются
- **Контролируйте скорость**: Используйте +/- для контроля скорости симуляции

## 🐛 Решение Проблем

**Q: Черный экран**
A: Убедитесь, что Panda3D установлен: `pip install panda3d`

**Q: Низкий FPS**
A: Это нормально при 30+ организмах. Камеру можно отодвинуть дальше.

**Q: Нет звука**
A: Звук не реализован в текущей версии (планируется)
