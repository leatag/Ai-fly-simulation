# 🌿 Реалистичный Цифровой Симулятор Организмов

## ✨ Новые Функции Эволюции

### 1. **Реалистичные Эмоции и Поведение**
Организмы теперь имеют настоящие эмоции:
- **Страх (Fear)** - животное пугается опасности и ищет убежище
- **Поиск (Searching)** - активный поиск пищи и воды при стрессе
- **Счастье (Happy)** - довольное существо с полным животом
- **Обычное (Normal)** - сбалансированное поведение

### 2. **Система Обучения Страху**
Организмы **учатся на опыте**:
- Если животное получает урон в месте - оно запоминает это как опасное
- Страх медленно развивается ассоциированием и теплеет с неудачами
- Животное избегает опасных мест на основе памяти
- Страх передается поблизости (рассеивается на соседних клетках)
- Страх постепенно забывается со временем

```python
# Пример: животное получает урон и запоминает место
org.record_danger_experience("attack")
# Теперь животное боится этого места
danger_level = org.brain.fear_memory.get_position_danger(x, y)
```

### 3. **Плавные Анимации**
Все движения животных теперь **плавные и реалистичные**:
- Организмы движутся между клетками с smooth interpolation (10 кадров)
- Крылья мух анимируются волнообразно
- Когда животное ест/строит - видна соответствующая анимация
- Тело мухи рисуется с 3D эффектом и сегментацией

### 4. **Красивые Модели Мух**
Реалистичные визуальные модели:
- **Нормальная муха**: спокойная, закрытые крылья, обычный цвет
- **Испуганная муха**: взволнованная, быстрое движение крыльев, красноватые глаза
- **Голодная муха**: активный поиск, зависит позиция и траектория к еде
- **Счастливая муха**: ленивые крылья, яркий цвет, закрытые довольные глаза


### 5. **Реалистичный Мир с Рельефом**
- Трава теперь рисуется с вариацией текстур
- Высота местности влияет на цвет (выше = темнее)
- Влажность влияет на зеленость травы
- Вода блестит и имеет световой эффект
- Каждая клетка имеет свою уникальную текстуру

### 6. **Более Умные Организмы**
Улучшенная система принятия решений:
- Животные помнят хорошие и плохие места
- Учитывают эмоциональное состояние при выборе действий
- Активнее исследуют когда сыты
- Когда голодны - целенаправленно ищут еду
- Избегают опасных зон на основе памяти страха

### 7. **Параметры Страха (Fear Learning)**
В `ai.py` есть конфигурация:
```python
FEAR_LEARNING_RATE = 0.15     # Скорость усвоения страха
FEAR_DECAY = 0.995            # Скорость забывания (0.995 = медленное)
MAX_FEAR = 1.0                # Максимальный уровень страха
DANGER_MEMORY_SIZE = 30       # Помнит последних 30 событий опасности
```

Чем выше `FEAR_LEARNING_RATE` - тем быстрее животное учится бояться.
Чем выше `FEAR_DECAY` - тем дольше помнит опасность.

## 🧬 Генетика + Эмоции

Поведение животного - комбинация **генов и эмоций**:
- Умное животное (GENE_INTELLIGENCE) быстрее учится страху
- Социальное животное (GENE_SOCIALITY) скапливается около других
- Агрессивное животное (GENE_AGGRESSION) атакует, если не боится
- Интеллект влияет на скорость обучения (LEARNING_RATE * intelligence)

## 💡 Примеры Наблюдений

### Сценарий 1: Обучение Страху
```
1. Муха летит мирно
2. Она получает урон от ловушки
3. Через несколько ударов - она запоминает место
4. В следующий раз она избегает этого места!
5. Со временем страх угасает, и она забывает об опасности
```

### Сценарий 2: Коллективное Поведение
```
1. Голодные мухи активно ищут еду (красное свечение, быстрые крылья)
2. Они находят источник и едят
3. После еды - становятся счастливыми (зеленый цвет)
4. Счастливые мухи строят убежища и размножаются
5. Потомки наследуют гены родителей (интеллект, память о страхе)
```

### Сценарий 3: Эволюция Поведения
```
1. Первое поколение - беспорядочное поведение
2. Со временем - умные животные выживают чаще
3. Страшливые животные избегают опасности
4. Умные + страшливые = лучше выживают
5. Они размножаются и передают гены!
```
